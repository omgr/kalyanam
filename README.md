## Getting Started

### Creating Your Wedding

1. Open Kalyanam in your browser
2. Click "Start Planning" on the landing page
3. Enter your wedding details:
   - Wedding name (e.g., "Priya & Rahul's Wedding")
   - Wedding date
   - Budget (optional)
   - Select your cultural tradition
4. Add the bride and groom names
5. Click "Create Wedding"

### Preview Cultural Templates

Before creating a wedding, you can preview cultural rituals:

1. Click "Browse cultural templates first" on the landing page
2. Select a culture to see all included ceremonies
3. Expand each ceremony to see details, significance, and required items
4. Use this to verify the template matches your traditions

### Logging In / Syncing to a New Device

If you already have wedding data on another device:

1. Click **"Login / Sync"** on the landing page
2. Choose your sync method:
   - **Enter Sync Code**: Get a 6-character code from the device with your data
   - **Import Backup File**: Upload a `.kalyanam.json` backup file
   - **Select Existing Wedding**: If you have weddings on this device already

#### Using Sync Code (Recommended for Remote Sync)

On the device WITH your data:
1. Go to Sync & Backup from the menu
2. Tap "Share from Here" 
3. Note the 6-character code displayed

On the new device:
1. Click "Login / Sync"
2. Select "Enter Sync Code"
3. Enter the 6-character code
4. Your wedding data transfers directly

### Privacy First

All your data is stored locally on your device. No account required, no cloud servers - your wedding details stay private.

## Events & Ceremonies

### Viewing Events

Navigate to **Events** from the sidebar to see all your ceremonies and events organized by date.

### Adding a New Event

1. Click "Add Event" button
2. Fill in the event details:
   - Event Name (e.g., "Mehndi Ceremony")
   - Local Name (optional, e.g., "మెహెందీ")
   - Date and Time
   - Venue and Address
   - Category (Pre-wedding, Wedding Day, Post-wedding)
   - Assign family members
   - Budget for this event
3. Add checklist items for the event
4. Click "Create Event"

### Cultural Templates

When you select a culture during setup, events are automatically populated with traditional ceremonies. For Telugu Brahmin weddings, this includes:

- Nischitartham (Engagement)
- Pellikuthuru/Pellikoduku
- Snathakam
- Kashi Yatra
- Muhurtham (Main Wedding)
- Talambralu
- Saptapadi
- Appaginthalu
- And many more...

### Managing Event Checklist

Each event can have a checklist of tasks:
- Click on an event to view details
- Add items to the checklist
- Check off items as they're completed
- Track progress with the visual progress bar

## Guest Management

### Adding Guests

1. Go to **Guests** from the sidebar
2. Click "Add Guest"
3. Enter guest details:
   - Name
   - Phone number
   - Email (optional)
   - Side (Bride's/Groom's)
   - RSVP Status
   - Plus-ones count
   - Dietary restrictions
   - Group/Table assignment

### RSVP Tracking

Filter guests by RSVP status:
- **Confirmed**: Guests who have confirmed attendance
- **Pending**: Awaiting response
- **Declined**: Cannot attend
- **Maybe**: Uncertain

### Guest Statistics

View real-time stats:
- Total invited guests
- Confirmed attendees (including plus-ones)
- Pending responses
- Declined invitations

## Budget & Expenses

### Setting Up Budget Categories

Categories help organize your spending:
- Venue & Decoration
- Catering & Food
- Photography & Video
- Clothing & Jewelry
- Music & Entertainment
- And more...

### Recording Expenses

1. Go to **Budget** from the sidebar
2. Click "Add Expense"
3. **Scan Receipt** (Optional):
   - Click "Take Photo" to capture a receipt
   - Or "Upload" to select from files
4. Fill in details:
   - Category
   - Description
   - Amount
   - Payment method (Cash, Card, UPI, Bank Transfer, Cheque)
   - Link to Vendor (optional)
5. **Payment Schedule** (for installments):
   - Toggle "Pay in Installments"
   - Set number of installments
   - Choose start date
   - Click "Generate Schedule"
6. Click "Add Expense"

### Payment Tracking

Track different payment types:

#### One-Time Payments
- Record the full amount
- Mark as paid when completed

#### Installment Payments
- Set up payment schedules
- Get automatic reminders before due dates
- Mark individual installments as paid
- Track partial payments

#### Stage-Based Payments
- Link payments to event milestones
- Pay when specific stages are completed

### Reconciliation

Match your expenses with bank transactions:
1. Go to Budget → Reconciliation tab
2. For each expense, enter the bank reference number
3. Click "Reconcile" to mark as verified

### Budget Analytics

- View spending by category
- Track budget utilization percentage
- See pending payments
- Monitor overdue amounts

## Task Management

### Creating Tasks

1. Go to **Tasks** from the sidebar
2. Click "Add Task"
3. Fill in:
   - Task title
   - Description
   - Priority (Low, Medium, High, Urgent)
   - Due date
   - Assign to family members
   - Link to event (optional)

### Task Priorities

- **Urgent**: Red flag, needs immediate attention
- **High**: Orange, important but not immediate
- **Medium**: Yellow, standard priority
- **Low**: Gray, can wait

### Task Status

- **Pending**: Not started
- **In Progress**: Currently working on
- **Completed**: Done
- **Cancelled**: No longer needed

## Sync & Backup

Kalyanam offers three free, privacy-focused ways to sync data between devices.

### Export/Import File

1. Go to **Sync & Backup** from the sidebar
2. Click **Export Data** to download your wedding as a JSON file
3. On another device, click **Import Data** to restore

### QR Code Sync

1. On Device A: Click **Show QR Code**
2. On Device B: Click **Scan QR Code**
3. Point camera at the QR code
4. Data transfers automatically

### P2P Real-time Sync

1. On Device A: Click **Share from Here**
2. Share the 6-character code with Device B
3. On Device B: Enter the code and click **Connect**
4. Data transfers directly between devices

See the [Sync Guide](sync-guide.md) for detailed instructions.

## Tips for Success

1. **Start Early**: Begin planning 6-12 months before the wedding
2. **Update Regularly**: Keep events, tasks, and budget current

4. **Track Payments**: Record all expenses as they happen

## Troubleshooting

### Data Not Loading
1. Refresh the page
2. Check if you're in the correct wedding (if multiple)
3. Clear browser cache and reload

### Lost Data
- Kalyanam stores data locally in your browser
- Clearing browser data will remove wedding data
- Use the export feature to backup regularly

### Performance Issues
- Close other browser tabs
- Clear browser cache
- Use a modern browser (Chrome, Firefox, Edge)
